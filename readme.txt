== Installation ==

1. Install the Live Shopping & Shoppable Videos Plugin from WooCommerce Marketplace.
2. After installing the plugin successfully, the next step is to configure the plugin settings.
3. To do so, sign up for a [Channelize.io account](https://channelize.io/lsc/account/login), choose a subscription plan, and complete the sign-up process.
4. You will then be redirected to your Live Shopping & Shoppable Videos Dashboard. Now go to your dashboard's Help & Support page, where you can see your Public & Private Keys as well as other account information.
5. Copy & paste these keys into the plugin settings and save changes.
6. Once this is completed, the Plugin has been successfully configured. You can find your Live Shop Page URL in the Plugin's Live Shop Page.
7. You are now ready to create Shopping Shows in your dashboard and go Live from broadcaster apps.
8. Please see the Instructions page in the Live Shopping & Shoppable Videos plugin for more information.

Note:

1. It is mandatory to install/activate WooCommerce to use the Live Shopping & Shoppable Videos plugin.
2. Our plugin is compatible with WooCommerce versions 4.0.1 and higher.
